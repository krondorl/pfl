# Photography Files Lister (pfl)

This CLI tool lists photography files from well known major brands:
- Canon
- Sony
- Nikon
- Fujifilm
- Panasonic

## Extension Set

### RAW Extensions (core)

```
.cr2
.cr3
.arw
.nef
.nrw
.raf
.rw2
```

### Compressed Photo Extensions

```
.jpg
.jpeg
.heif
.heic
```
